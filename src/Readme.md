# src Folder

This folder contains all frontend code for the Fleet Management System, built with React and TypeScript.

- **components/**: Modular UI components for each core system page:
  - **Analytics/**: Operational analytics and financial reports.
  - **Auth/**: Login, registration, and OTP verification forms.
  - **Dashboard/**: Command center for high-level fleet oversight and KPIs.
  - **Drivers/**: Driver management, compliance, and performance tracking.
  - **Expenses/**: Expense and fuel logging forms and tracking.
  - **Layout/**: Navigation and sidebar components.
  - **Maintenance/**: Maintenance and service log management.
  - **Trips/**: Trip dispatcher, trip creation, and management.
  - **Vehicles/**: Vehicle registry and asset management.
- **contexts/**: React context for authentication and global state.
- **lib/**: API client and utility functions for backend communication.

**Purpose:**
Implements all user interfaces, forms, dashboards, and workflows for the digital fleet management hub.
