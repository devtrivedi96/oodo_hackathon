# Backend

Node.js backend for the project. Responsible for API endpoints, database access and authentication.

Contents:
- `.env` – environment variables (local)
- `server.js` – backend entry point
- `databse.js` – DB helper (note: file name in repo)
- `package.json` – backend dependencies and scripts
- `schema.sql` – SQL schema for the project's database
- `config/` – database config utilities
- `server/` – organized server code: controllers, routes, middleware, utils

Run and configure the backend from this folder.
