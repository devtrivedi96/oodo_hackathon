# Server Folder

Contains main server logic, including API routes, controllers, middleware, and utilities.

- **config/**: Server DB config.
- **controllers/**: Business logic for auth, users, trips, vehicles, etc.
- **middleware/**: Express middleware (auth, error handling).
- **routes/**: API route definitions.
- **utils/**: Utility functions (email, helpers).

**Purpose:**
Implements backend workflows and API endpoints.
