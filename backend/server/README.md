# Server (backend/server)

Structured server-side code for handling routes, controllers and middleware.

Subfolders:
- `config/` – lower-level server config (DB wrappers)
- `controllers/` – request handlers (e.g., `authController.js`)
- `middleware/` – express middleware (e.g., `authMiddleware.js`)
- `routes/` – route definitions (e.g., `authRoutes.js`)
- `utils/` – utilities (e.g., `sendEmail.js`)

Use this folder to explore the app-level API logic.
