# Frontend (`src`)

React + TypeScript app entry and components.

Top-level files:
- `main.tsx` – React entry mounting `App`
- `App.tsx` – top-level app component and routes
- `index.css` – global styles
- `vite-env.d.ts`, `tsconfig.*` – TypeScript config files

Subfolders:
- `components/` – UI components and feature areas
- `contexts/` – React contexts (e.g., `AuthContext`)
- `lib/` – shared libraries (e.g., `db.ts` helper)
